## 📘 Project Description
This is a Selenium WebDriver automation framework for the OrangeHRM application using the Page Object Model (POM) design pattern in Java with TestNG and Maven. It covers end-to-end HR workflows including employee management, leave application, and system configuration.

---

## 🔧 Project Structure
```
OrangeHRM_Testing_Automation_Suite/
├── pom.xml
├── src/
│   └── test/
│       └── java/
│           ├── test_cases/
│           └── webpage_ui_locators/
```

---

## 🧪 Features Covered
- Admin adds new employees and assigns leave entitlements.
- Admin approves leaves and changes passwords.
- New users apply for leave and verify their info.
- Admin enables/disables application modules.
- TestNG framework with Page Object Model (POM).

---

## 🛠 Tools Used
- Java 17
- Selenium WebDriver 4.33.0
- TestNG 7.11.0
- Maven
